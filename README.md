## How It Works

The system is built in layers:

1. **Ingestion Layer** – connects all company data sources
2. **Knowledge Layer** – structures and cleans information
3. **Memory Layer** – builds entities, relationships, and timelines
4. **Retrieval Layer** – enables intelligent, grounded search
5. **Intelligence Layer** – tracks commitments, risks, and signals
6. **Interface Layer** – provides dashboards, search, and chat
7. **Security Layer** – ensures enterprise-grade trust and control
